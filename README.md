# monoМаркет — checkout prototype

Кликабельный прототип чекаута monoМаркет. Чисто статика — без сборки и без бэка.

## Що всередині

- `index.html` — Кошик
- `checkout.html` — Чекаут (Ваше замовлення) з табами Частинами/Повністю та Відділення/Кур'єр + bottom sheet вибору картки
- `delivery.html` — Список відділень Нової пошти з пошуком
- `address.html` — Список адрес для кур'єрської доставки
- `recipient.html` — Список отримувачів
- `success.html` — Екран успіху з mono-котом
- `styles.css` — design system + всі компоненти
- `app.js` — стейт у `localStorage`, утиліти та SVG-іконки

## Як подивитися

Відкрий `index.html` у браузері — і все. Локально без сервера працює.

Або задеплой на GitHub Pages — і ділись посиланням.

## Прототип, не продакшн

Стейт у `localStorage` — у різних людей буде свій. Жодних API. Кнопка «Сплатити» імітує processing 1.4с і веде на success.
